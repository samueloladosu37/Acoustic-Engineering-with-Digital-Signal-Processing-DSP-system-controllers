## Wisdom Audio SC-2 & SC-3 — Second Generation System Controllers

Fully calibrated hi-resolution audio controllers for home theater 
and multi-room applications.

### Specifications

| Parameter         | SC-2                        | SC-3                        |
|-------------------|-----------------------------|-----------------------------|
| I/O Configuration | 8-in / 16-out               | 16-in / 32-out              |
| Dirac Outputs     | 16 channels                 | 16 channels                 |
| Rack Units        | 2RU                         | 3RU                         |
| Analog I/O        | Balanced XLR (all channels) | Balanced XLR (all channels) |
| Processing        | 24-bit / 96 kHz             | 24-bit / 96 kHz             |
| SNR               | −110 dB                     | −110 dB                     |
| Configuration     | Ethernet / IP HTML page     | Ethernet / IP HTML page     |

### Key Features
- **Dirac Live Room Correction Suite™** with multi-zone capability
- **Expert Mode** for advanced live tuning
- Model-specific Wisdom Audio speaker crossover and PEQ profiles
- Fully adjustable active crossovers, peaking filters, and delays
- Up to **16 independently calibrated zones** for distributed music 
  or multichannel theater
- **4-preset slots** for complete independent calibrations, 
  recallable via RS-232 (AMX, Control4, Crestron, Savant compatible)
- Browser-based configuration over Ethernet

### Setup Workflow
1. Connect SC unit to local network via Ethernet
2. Access IP-based HTML configuration page via browser
3. Connect USB microphone to laptop; connect laptop wirelessly to SC
4. Run Dirac Live calibration using Wisdom Audio software suite
5. Store calibration to preset slot; assign zone and recall via 
   control platform (RS-232)
