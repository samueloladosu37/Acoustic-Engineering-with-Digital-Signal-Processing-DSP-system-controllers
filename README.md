# Acoustic Systems & Audio DSP — Technical Notes

# SC-2 & SC-3 System Controllers

## Overview

The **SC-2** and **SC-3** are professional Digital Signal Processing (DSP) system controllers developed by Wisdom Audio for high-performance home theater, cinema and multimedia-room audio installations. They serve as the central processing unit of an audio system by managing signal routing, crossover processing, equalization, delay, room correction, and speaker optimization before audio reaches the power amplifiers.

Unlike a conventional audio processor, a system controller ensures that every loudspeaker receives a signal specifically optimized for its frequency range and acoustic environment.

---

## System Role in the Audio Signal Chain

A typical professional signal chain is:

```text
Audio Source
      │
      ▼
AV Processor / Mixer
      │
      ▼
SC-2 / SC-3 System Controller (DSP)
      │
      ▼
Power Amplifiers
      │
      ▼
Loudspeakers
```

The system controller sits between the source processor and the power amplifiers, acting as the "brain" of the audio system.

## Why a System Controller is Important

A system controller is one of the most important devices in a professional audio installation because it ensures that every speaker performs as it was designed.

### 1. Room Correction

Using **Dirac Live** - a unique feature, the controller compensates for room reflections, standing waves, and acoustic irregularities, producing a flatter and more accurate frequency response.

### 2. Active Crossovers

The controller divides audio into low, mid, and high frequencies and sends each frequency band to the correct amplifier and loudspeaker driver.

### 3. Time Alignment

Different loudspeakers are positioned at different distances from listeners. Delay processing synchronizes all speakers so sound arrives simultaneously.

### 4. Equalization

Built-in Parametric EQ (PEQ) allows engineers to correct unwanted frequency peaks and dips without changing the character of the loudspeaker.

### 5. Speaker Protection

Proper crossover settings, filters, and output limiting help protect expensive loudspeakers from excessive low frequencies and signal overload.

### 6. Multi-Zone Audio

The SC-2 and SC-3 can independently calibrate up to sixteen listening zones, making them suitable for distributed audio systems.

### 7. Preset Management

Multiple calibration profiles can be stored and recalled instantly for different room layouts or listening modes.

# Advantages of Using a System Controller

Using a DSP system controller provides several benefits:

- Improved sound clarity and accuracy
- Better bass response
- Accurate speaker crossover management
- Precise time alignment between speakers
- Professional room calibration using Dirac Live
- Increased loudspeaker protection
- Easier troubleshooting and system configuration
- Centralized control of multiple speaker zones
- Consistent sound quality throughout an installation
- Reduced installation time through software configuration

---

# Disadvantages of Not Using a System Controller

Without a system controller, several performance issues can occur:

- Poor frequency balance
- Uneven bass response
- Phase cancellation between speakers
- Incorrect crossover points
- Loss of stereo imaging
- Reduced speech intelligibility
- Increased risk of loudspeaker damage
- Inconsistent sound across listening positions
- Difficult manual system tuning
- Inability to compensate for room acoustics
